# Relationship Nurture Assistant 亲密关系滋养助手

帮你处理伴侣、亲子、代际关系中的沟通难题——从吵架降温到长期关系滋养，提供实用话术和行动方案。

## What it does
- 识别关系类型和核心冲突模式
- 生成具体沟通话术和"我开头"的陈述改写
- 设计结构化的关系修复和滋养计划
- 覆盖夫妻、异地恋、亲子等常见关系场景

## Example scenarios

**1. 家务分工吵架**
> 👤 我和伴侣总为家务分工吵架，怎么打破这个循环？
> 🤖 先识别深层需求——是渴望被认可、追求公平、还是需要掌控感？然后用"我开头"改写：不说"你总是不洗碗"，改成"我一个人做家务时会觉得很累，希望我们可以一起分担。" 建议建立一个可视化任务清单 + 每周轮换制，每周核对一次，冲突升级前先停 5 分钟再做回应。

**2. 结婚多年感觉像室友**
> 👤 结婚 10 年感觉像室友，怎么重建情感亲密？
> 🤖 亲密重建不是一蹴而就的，可以用"脆弱性阶梯"——从低风险分享开始（今天工作中一件有趣的事），逐步到中等分享（我的一个担忧），再到深层分享（我对我们关系的感受）。同时每周安排一次"新鲜体验"（一起尝试新餐厅、新运动、新课程），这能在神经化学层面触发联结感。建议按 6 周计划推进，每周一个小目标。

**3. 异地恋怎么维持感情**
> 👤 我和女朋友异地，她在杭州我在北京，一个月见一次，每天就是"吃了没""下班了"，怎么办？
> 🤖 异地需要"创造共同经历"——日常：每天固定 20 分钟视频，轮流选活动（一起看同一部剧讨论、玩线上桌游、同步学一门课交流心得）；每周：寄一个实物（零食/书/手写信），用快递代替见面；每月：用同一个周末去各自城市的同一主题餐厅/展览，保持共同话题；每季度：轮流飞去对方城市过周末，提前规划要做的新事情。关键是创造内容，不是靠查岗维持存在感。
